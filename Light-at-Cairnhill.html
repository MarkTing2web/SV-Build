The Challenge: Unifying Fragmented Systems for Private Lift Security

Light@Cairnhill, a 121-unit luxury condominium at Cairnhill Circle, faced the challenge of modernizing two separate aging systems while addressing the unique security requirements of private lift access — where elevators open directly into residential units rather than common corridors.

Fragmented Legacy Systems

The property operated two distinct systems that had aged independently. The Fermax intercom system handled visitor communication at lobby entrances, while separate biometric fingerprint readers controlled building access at the same locations. This fragmentation created operational complexity — residents and visitors needed to interact with two different devices at each entry point, and management maintained two separate systems with different vendor support requirements and no integration between them.

The dual-system approach also created confusion during visitor arrivals. Visitors would use the Fermax intercom to call residents, but then needed separate instructions to use the fingerprint reader for actual door release — a clunky user experience that felt outdated compared to modern unified access control systems.

Private Lift Access Security Requirements

Unlike conventional condominiums where lifts open to common corridors serving multiple units, Light@Cairnhill features private lift access — elevators that open directly into individual homes. This architectural configuration elevates security requirements significantly. A compromised lift access control system could allow unauthorized individuals to ride elevators that open directly into residents' private living spaces, bypassing traditional unit door security entirely.

The existing lift access system lacked the sophisticated controls needed for this higher-security environment. Management needed granular control over which residents could access which specific lifts, time-based access restrictions for domestic helpers and service providers, and comprehensive audit trails showing exactly who accessed private lifts and when. The legacy system offered none of these capabilities.

Infrastructure Constraints and Resident Impact

As with L'viv, the building's original construction used 2-wire analog cabling for the intercom system. Modern IP-based video intercom systems typically require Cat6 Ethernet cabling, and running new cables to 121 occupied units would necessitate extensive renovation work — ceiling panel removal, wall penetration, and significant resident disruption over weeks or months.

The MCST needed to balance system modernization with resident convenience. Any solution required minimizing disruption to occupied units while delivering the enhanced security and mobile connectivity that modern luxury residents expected.

Optional Equipment Strategy

Adding complexity to the upgrade planning, not all residents required the same level of system integration. While building-level access control (lobbies, lifts, gates) needed complete replacement for security and operational reasons, indoor video monitors represented an optional convenience feature. Some residents preferred the convenience of in-unit video screens for visitor verification, while others were content relying solely on the mobile app for visitor management.

The MCST required a solution that could deliver comprehensive building security upgrades while accommodating resident choice on optional equipment, avoiding the expense of forcing uniform indoor monitor installations across all 121 units when resident preferences and budgets varied.

Our Solution: Unified Access Control with Private Lift Security

Securevision designed a comprehensive upgrade replacing both the Fermax intercom and separate fingerprint readers with a unified Akuvox video intercom ecosystem, while implementing sophisticated lift access control tailored for private elevator security requirements.

Unified Video Intercom Replacing Dual Systems

We deployed 27 Akuvox X915 video visitor panels across both lobby systems (Lobby A with 14 panels and Lobby B with 13 panels), completely replacing the previous Fermax intercom units and eliminating the separate fingerprint readers entirely. Each X915 panel consolidates what previously required two separate devices into a single elegant touchscreen interface.

The 8-inch X915 panels transform the visitor experience from basic audio communication to high-definition video calls with integrated biometric access control. Residents enrolled in the system can use face recognition for hands-free building entry, eliminating the need to carry and present access cards. The panels also support RFID cards, PIN codes, and QR code access via the SmartPlus mobile app, providing multiple authentication methods through one unified device.

Beyond the lobby systems, we installed 2 additional X915 panels at the side and rear gates, ensuring consistent access control methodology across all building entry points. Visitors experience the same intuitive interface whether arriving at the main lobbies or service entrances, while management maintains centralized control over all access points through a single platform.

The 2-Wire Infrastructure Innovation

Rather than undertaking expensive Cat6 rewiring throughout the building, we leveraged Akuvox's NS-2 2-wire converter technology — the same cost-saving approach successfully deployed at L'viv. We installed 20 NS-2 units throughout Light@Cairnhill's common area risers, with each unit supporting up to 6 connected devices.

The NS-2 serves dual functions: converting existing 2-wire analog cabling to carry IP network data, while simultaneously providing power to connected devices over the same 2-wire infrastructure. This solution delivered modern IP video intercom capability without the massive expense and resident disruption of complete rewiring across 121 occupied units.

Optional Indoor Monitors — Resident Choice

Recognizing that resident preferences and budgets varied, the MCST adopted a flexible approach to indoor video monitors. Rather than mandating uniform installations across all units, residents could choose whether to purchase Akuvox indoor monitors for in-unit video calling capability.

Residents who opted for indoor monitors receive 7-inch touchscreen displays showing high-definition video when visitors arrive at entrance panels. Those who preferred not to install monitors could rely entirely on the SmartPlus mobile app for visitor management — answering calls on their smartphones and granting access remotely without requiring additional hardware installations.

This optional equipment strategy reduced overall project costs while respecting resident autonomy. The building-level infrastructure supports both scenarios seamlessly — residents with monitors and residents using mobile-only access operate within the same unified system.

Private Lift Access Control — Enhanced Security

The most critical security component addressed Light@Cairnhill's private lift configuration. We implemented 8 Akuvox EC33 elevator controllers — 6 units for the main lobby lifts serving residential floors, plus 2 units for service lifts.

The EC33 controllers integrate directly with the lift system machinery, enabling sophisticated access control that goes far beyond traditional floor-button restrictions. When residents authenticate at lobby X915 panels (via face recognition, card, PIN, or mobile app), the EC33 controller receives their credential information and determines which specific lift they can access and which floor button becomes active.

This granular control is essential for private lift security. Unlike conventional buildings where any authorized resident can access any lift and press any floor button, Light@Cairnhill's system ensures residents can only access lifts that serve their specific unit and can only activate the button for their designated floor. This prevents scenarios where residents could inadvertently or intentionally access lifts opening directly into other residents' homes.

The EC33 integration also supports time-based access management. Domestic helpers, service providers, or temporary occupants can receive credentials that work only during specified hours or days, with automatic expiration eliminating the need for manual credential revocation. All lift access events log automatically, creating comprehensive audit trails showing exactly who accessed private lifts and when — critical for security investigations if incidents occur.

Amenity Access Control

Beyond lobbies and lifts, we installed 1 Akuvox A05S biometric reader at the gymnasium. This commercial-grade access control terminal uses the same credential database as the lobby panels, allowing residents to access shared amenities using their enrolled faces, RFID cards, or PIN codes without requiring separate gym-specific credentials.

The unified credential approach simplifies resident experience — one face enrollment, one card, or one PIN works across building entry, private lifts, and amenity access. Management benefits from consolidated user administration rather than maintaining separate databases for different facility types.

Guard Portal for Centralized Monitoring

We implemented the Akuvox VP-R49G guard portal at the security post, creating a centralized dashboard where guards monitor all 27 entrance panels (both lobbies plus gates) simultaneously. The 7-inch Android-based touchscreen displays real-time visitor calls, allows manual access override when residents are unreachable, and maintains comprehensive logs of all visitor and access events.

When residents issue temporary digital keys through the SmartPlus mobile app, the guard portal receives notifications showing which visitors have pre-authorized access, reducing guard workload during high-traffic periods. Guards can also create time-limited digital credentials directly from the portal for contractors or service providers requiring temporary building access.

SmartPlus Mobile App Integration

The SmartPlus mobile application unifies all access functions into a single resident-facing interface accessible on iOS and Android devices. Residents answer visitor calls from anywhere — whether at home, in the office, or traveling internationally — viewing live HD video and granting or denying access with a single tap.

The app supports time-limited temporary key generation with granular controls. Residents specify exact validity windows for recurring services (like "weekdays 9 AM-12 PM for the next month" for cleaning services) or one-time access (like "today 2-4 PM" for furniture delivery). All access events automatically log in the app's activity history, providing residents complete audit trails for security and household management.

Each household received SmartPlus app licenses for up to 4 smartphones, enabling families to distribute access management across multiple family members while maintaining centralized visibility of all access events.

System Integration Philosophy

The unified Akuvox ecosystem consolidates what previously required multiple separate systems — Fermax intercom, biometric fingerprint readers, lift access control, and amenity access — into a single integrated platform. Residents, management, and guards all interact with the same system architecture, simplifying training, support, and troubleshooting while enabling capabilities impossible with fragmented legacy systems.

Private Lift Security Logic

The EC33 controllers are the critical security layer for Light@Cairnhill's direct-to-home lift configuration. Unlike conventional buildings, these controllers interface with the lift machinery to ensure that when a resident is authenticated (via face, card, or app), only the specific lift serving their unit is called, and only their designated floor button is enabled. This prevents unauthorized residents or visitors from accidentally accessing elevators that open directly into private living rooms.

SmartPlus Mobile Integration

The entire 121-unit estate is unified via the SmartPlus cloud platform. Residents can answer video calls from any entrance panel globally, issue time-limited digital keys for service providers, and review access logs for their specific household. The platform supports up to 4 mobile instances per unit, allowing complete family-wide access management.
